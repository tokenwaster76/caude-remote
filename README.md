# Windows Remote MCP

Control your Windows PC from the **Claude mobile app** using the Model Context Protocol (MCP).

## What it does

Runs a local MCP server on your Windows machine and exposes it over a secure Cloudflare tunnel so the Claude app on your phone can:

| Tool | Description |
|---|---|
| `execute_command` | Run shell commands (cmd.exe) |
| `read_file` | Read any allowed file |
| `write_file` | Write / create files |
| `list_directory` | Browse the file system |
| `get_system_info` | CPU, RAM, uptime, processes |
| `take_screenshot` | Capture the desktop |

## Quick start

### 1. Prerequisites

- [Node.js 20+](https://nodejs.org)
- A [Cloudflare account](https://cloudflare.com) (free tier works)

### 2. Setup

```powershell
git clone <this-repo>
cd caude_remote
.\scripts\setup.ps1
```

This installs dependencies, builds TypeScript, creates a `.env` with a random `AUTH_TOKEN`, and downloads `cloudflared.exe`.

### 3. Configure `.env`

Edit `.env` and set `ALLOWED_PATHS` to the directories you want the server to access:

```env
AUTH_TOKEN=<auto-generated — keep this secret>
PORT=3000
ALLOWED_PATHS=C:\Users\Mario
ALLOW_SHELL_OPERATORS=false
```

### 4. Start

```powershell
.\scripts\start-tunnel.ps1
```

The script prints a tunnel URL like:

```
>>> Tunnel URL: https://abc-123.trycloudflare.com/mcp
>>> Bearer token is in your .env file (AUTH_TOKEN)
```

### 5. Connect Claude mobile

1. Open **Claude** on your phone
2. Go to **Settings > MCP Servers > Add Server**
3. Enter:
   - **URL**: `https://abc-123.trycloudflare.com/mcp`
   - **Auth**: Bearer `<your AUTH_TOKEN>`

That's it — Claude can now control your PC.

## Auto-start on login (optional)

Run once as Administrator:

```powershell
.\scripts\install-service.ps1
```

Creates a Windows Scheduled Task that starts the server + tunnel automatically when you log in.

## Security notes

- The `AUTH_TOKEN` is the only thing protecting your machine — keep it secret and make it long.
- `ALLOW_SHELL_OPERATORS=false` (default) blocks pipes and chaining to limit blast radius.
- `ALLOWED_PATHS` restricts which files/directories the server can touch.
- The Cloudflare tunnel uses HTTPS; traffic never touches your firewall/router.

## Architecture

```
Claude mobile app
      |  HTTPS
      v
Cloudflare Tunnel  (cloudflared.exe)
      |  HTTP (localhost)
      v
Express MCP server  (src/server.ts)
      |  Bearer auth + session management
      v
MCP tools  (execute_command, read_file, ...)
      |
      v
Windows OS
```
